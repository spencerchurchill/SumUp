Sumit Manual

Sumit will work regardless of the Aylien API; however, without the API, the extension will load Iframes as previews instead of summaries.

1. Create Account

First, you need to go to Aylien to create an account.

2. Open Dashboard

Next, activate your account (by checking your email) and proceed to the Dashboard.

3. Use App ID and API key

Under the Credentials box, note the App ID and API key. Simply input the App ID and API key into their respective input boxes in the Sumit Popup.

4. Enable Sumit

In the Whitelist textbox, add websites you wish to enable Sumit on. If you wish to enable Sumit on every site, type *.


You are now ready to take advantage of all Sumit has to offer!
